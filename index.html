Тема Концепция Синтаксис / Пример Для чего нужно?
Переменные и типы данных Переменная
name = "Вася"
age = 25
 Именованная область памяти для хранения данных. Позволяет хранить и использовать данные в программе.
Переменные и типы данных Основные типы данных
int (целое): 10
float (дробное): 10.5
str (строка): "Привет"
bool (логический): True, False
 Определять, какого рода данные мы храним и что с ними можно делать.
Переменные и типы данных Динамическая типизация
x = 5 # x - int
x = "пять" # теперь x - str
 Тип переменной определяется автоматически. Позволяет легко менять тип данных переменной.
Основные операции Арифметические операции
+, -, *, /, //, %, **
result = 10 + 5 * 2 # 20
 Выполнять математические вычисления.
Основные операции Операции сравнения
==, !=, >, <, >=, <=
5 > 3 # True
"abc" == "abc" # True
 Сравнивать два значения. Возвращают True или False.
Основные операции Логические операции
and, or, not
age >= 18 and age <= 65
not is_weekend
 Комбинировать условия. Используются в условиях if и циклах.
Структуры данных Список (List)
my_list = [1, 2, 3, "текст"]
my_list.append(4) # [1, 2, 3, "текст", 4]
 Упорядоченная изменяемая коллекция. Может содержать элементы разных типов.
Структуры данных Кортеж (Tuple)
my_tuple = (1, 2, 3)
coordinates = (10, 20)
 Упорядоченная неизменяемая коллекция. Быстрее списков, используется для постоянных данных.
Структуры данных Словарь (Dictionary)
my_dict = {"ключ": "значение", "age": 25}
person = {"name": "Анна", "age": 30}
 Неупорядоченная коллекция пар "ключ-значение". Для хранения данных с быстрым доступом по ключу.
Структуры данных Множество (Set)
my_set = {1, 2, 3, 3, 4} # {1, 2, 3, 4}
unique_numbers = set([1, 2, 2, 3]) # {1, 2, 3}
 Неупорядоченная коллекция уникальных элементов. Удаляет дубликаты автоматически.
Управление потоком Условные операторы (if/elif/else)
if условие: действие elif другое_условие: другое_действие else: действие_по_умолчанию
if temperature > 30: print("Жарко") elif temperature > 20: print("Тепло") else: print("Прохладно")
 Выполнять разные блоки кода в зависимости от условий.
Управление потоком Цикл for
for элемент in последовательность: действие
for i in range(5): print(i) # 0, 1, 2, 3, 4
 Повторять блок кода для каждого элемента в последовательности.
Управление потоком Цикл while
while условие: действие
count = 0 while count < 5: print(count) count += 1
 Повторять блок кода, пока условие истинно.
Функции Определение функции
def имя_функции(параметры): тело_функции return результат
def greet(name): return f"Привет, {name}!"
 Создавать повторно используемые блоки кода. Упрощают программу и избегают повторения.
Функции Параметры по умолчанию
def функция(параметр=значение_по_умолчанию):
def power(x, n=2): return x ** n power(5) # 25 power(5, 3) # 125
 Позволяют вызывать функцию с меньшим количеством аргументов.
Объектно-ориентированное программирование (ООП) Класс и объект
class ИмяКласса: def __init__(self, параметры): self.атрибут = параметр def метод(self): действие
class Dog: def __init__(self, name): self.name = name def bark(self): return "Гав!"
 Создавать собственные типы данных, объединяющие данные и методы для работы с ними.
ООП Наследование
class ДочернийКласс(РодительскийКласс):
class Animal: def speak(self): pass class Cat(Animal): def speak(self): return "Мяу!"
 Позволяет создавать новый класс на основе существующего, наследуя его атрибуты и методы.
Обработка исключений Try/Except
try: потенциально_опасный_код except ТипОшибки: обработка_ошибки
try: number = int(input("Введите число: ")) except ValueError: print("Это не число!")
 Обрабатывать ошибки, чтобы программа не аварийно завершалась.
